**Payment Reminder Email Automation using n8n**

**Project Overview**

This project automates payment reminder emails for customers with upcoming or overdue invoices using n8n.

Instead of manually checking payment records and sending reminder emails, the workflow automatically:

- Reads payment data
- Filters pending invoices
- Generates reminder emails
- Sends personalized emails
- Logs workflow execution


**Business Problem**

Finance teams spend significant time manually tracking payment due dates and sending reminder emails.

This workflow reduces manual effort by automating the complete reminder process.

**Workflow**

Trigger
        ↓
Read Payment Data
        ↓
Filter Due Payments
        ↓
Generate Email
        ↓
Send Email
        ↓
Success Log
